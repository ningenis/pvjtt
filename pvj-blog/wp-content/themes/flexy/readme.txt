================================= Installation ================================

1. Automatic Installation:
      Go to WP-Admin > Appearance > Themes > Install Themes > Upload and choose the theme zip folder.

Manual installation:

1. Unzip flexy.Zip and Upload the `flexy` folder to the `/wp-content/themes/` directory

Activiation and Use
.....................

1. Activate the Theme through the 'Themes' menu in WordPress 
2. Click Appearance to get different options.
